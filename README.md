# qig_agent
构建原因：
 1. 聚焦工程落地痛点：工业产线中，批量生成符合业务规范的高质量封面图、设计素材等需求高频存在。**仅依赖人工编写提示词 + 直接调用生图模型的模式，难以兼顾批量生产、品类多样性、输出质量稳定性、低试错成本（不抽卡）与高准入率，因此需搭建专属Agent，形成标准化生图解决方案，高效满足工业级生图诉求。**
2. 赋能模型迭代与评测：**为后续生图模型的优化训练提供高质量、合规化的标注训练数据，同时构建符合业务场景要求的标准化评测集，**为模型性能验证与迭代提供可靠的数据支撑。
